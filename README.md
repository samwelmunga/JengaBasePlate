# JengaBasePlate

| Dependency | Version |
|---|---|
| Svelte | 5.53.7 |
| Capacitor | 6.2.1 |
| Tailwind CSS | 4.2.1 |

A template project using Svelte + Capacitor + Tailwind CSS, intended as a starting point for new projects.

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Capacitor

After building, sync with native platforms:

```bash
npx cap add android   # or ios
npx cap sync
npx cap open android  # or ios
```
